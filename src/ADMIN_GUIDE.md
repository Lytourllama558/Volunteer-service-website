# 后台管理系统使用指南

## 🔐 访问后台

### 方法一：通过导航栏
1. 在网站顶部导航栏点击 **"管理后台"** 按钮
2. 进入登录页面

### 方法二：直接访问
在浏览器地址栏直接访问后台登录页面

## 🔑 登录信息

**演示账号：**
- 用户名: `admin`
- 密码: `admin123`

> 注意：这是演示账号，生产环境中应使用更安全的认证机制

## 📊 功能模块

### 1. 仪表板 (Dashboard)

进入后台后的首页，提供整体数据概览：

**数据卡片：**
- 📋 志愿机会总数
- 👥 报名总数  
- ❤️ 可用名额/总名额
- 🎯 已占用名额

**快速视图：**
- 最新志愿机会列表（最近5条）
- 最新报名记录（最近5条）

### 2. 志愿机会管理

完整的志愿机会 CRUD 功能：

#### 添加新机会
1. 点击 **"添加新机会"** 按钮
2. 填写表单信息：
   - 标题 *（必填）
   - 组织名称 *（必填）
   - 分类 *（下拉选择）
   - 地点 *（必填）
   - 日期 *（必填）
   - 时长 *（必填）
   - 可用名额 *（数字）
   - 总名额 *（数字）
   - 描述 *（必填）
   - 要求（每行一条）
   - 标签（逗号分隔）
   - 图片 URL

3. 点击 **"创建"** 保存

#### 编辑机会
1. 在列表中找到要编辑的机会
2. 点击蓝色的 **编辑图标** 📝
3. 修改信息
4. 点击 **"保存修改"**

#### 删除机会
1. 在列表中找到要删除的机会
2. 点击红色的 **删除图标** 🗑️
3. 确认删除

#### 机会列表显示
- 标题
- 组织
- 分类（带颜色标签）
- 日期
- 名额（可用/总数）
- 操作按钮

### 3. 报名记录管理

查看和管理所有用户报名：

#### 功能特点
- **搜索功能**：按姓名、邮箱或电话搜索
- **筛选功能**：按志愿机会筛选
- **详细信息显示**：
  - 姓名
  - 联系方式（邮箱和电话）
  - 报名的志愿机会
  - 报名时间
  - 留言内容

#### 取消报名
1. 找到要取消的报名记录
2. 点击红色的 **删除图标** 🗑️
3. 确认取消
4. 系统会自动恢复一个可用名额

## 💾 数据存储

所有数据存储在 Supabase 数据库中：

### 数据表结构

**志愿机会表** (`opportunity:*`)
```
- id: 唯一标识符
- title: 标题
- organization: 组织名称
- category: 分类
- location: 地点
- date: 日期
- duration: 时长
- spotsAvailable: 可用名额
- totalSpots: 总名额
- description: 描述
- requirements: 要求数组
- image: 图片URL
- tags: 标签数组
- createdAt: 创建时间
```

**报名记录表** (`registration:*`)
```
- id: 唯一标识符
- opportunityId: 关联的志愿机会ID
- name: 姓名
- email: 邮箱
- phone: 电话
- message: 留言
- registeredAt: 报名时间
```

点击右上角的 **"退出登录"** 按钮，返回前台网站

## ⚠️ 注意事项

1. **数据验证**：
   - 名额数字必须大于0
   - 可用名额不能大于总名额
   - 必填字段不能为空

2. **删除操作**：
   - 删除志愿机会前请确认
   - 删除后数据无法恢复
   - 删除机会不会删除相关报名记录

3. **权限管理**：
   - 当前版本使用简单的账号密码验证
   - 建议在生产环境中使用更安全的认证方式

4. **性能优化**：
   - 表单数据会实时保存到数据库
   - 列表数据支持实时刷新
   - 大量数据时建议使用搜索和筛选

## 🚀 快速操作流程

### 发布新的志愿机会
1. 登录后台 → 志愿机会管理
2. 点击"添加新机会"
3. 填写完整信息
4. 保存 → 前端立即显示

### 查看某个机会的报名情况
1. 登录后台 → 报名记录管理
2. 在筛选下拉框选择对应的志愿机会
3. 查看该机会的所有报名记录

### 管理名额
1. 方法一：编辑志愿机会，直接修改名额数字
2. 方法二：取消报名会自动恢复名额
3. 仪表板实时显示总名额统计

## 💡 最佳实践

1. **定期检查**：每天登录后台查看新的报名记录
2. **及时更新**：活动结束后及时关闭或删除机会
3. **数据备份**：重要数据定期导出保存
4. **联系志愿者**：通过报名记录中的联系方式与志愿者沟通
5. **监控名额**：关注热门活动的名额情况，及时增加
